STATS RESIDUAL BOXPLOTS Extension Command

Plot a set of linked boxplots of the residual against predicted values for results from REGRESSION or a similar procedure. This is useful in checking for heteroscedasticity and functional form problems. The command does not run the regression itself.

STATS RESIDUAL BOXPLOTS
DEPENDENT=dependent variable*
PREDVALS varname*
RESIDUALS=varname*
LOESS=YES or NO**
NUMBINS=number

/HELP

* Required
** Default

STATS RESIDUAL BOXPLOTS /HELP. prints this information and does nothing else.


STATS RESIDUAL BOXPLOTS DEPENDENT=salary
PREDVALS=DRE_1 RESIDUALS=ADJ_1 
LOESS=yes.

There is an article on this procedure that is installed with the extension. It is placed in the STATS_RESIDUAL_BOXPLOTS directory under the location where the extension is installed.

SPLIT FILES and weights are not supported by this procedure.

DEPENDENT specifies the name of the dependent variable. Only its name is used.

RESIDUALS specifies the name of the preexisting residual values variable.

PREDVALS specifies the name of the preexisting predicted values variable.

LOESS optionally specifies whether or not to draw loess fit lines for the quintiles of the residuals.

NUMBINS optionally specifies the number of boxplots to be drawn. It must be between 3 and 30. By default, the number of bins is determined by the sample size.
